# powerhight
功能：  
* 杀软检测
* 补丁查询
* 网络信息枚举
* 注册表查询
* 脆弱服务枚举
* 用户信息与进程信息等获取

function:
```
---------------------------------------
|Supporting tools for power withdrawal|
|--------------------------------------
|Author:Jiushi| アクセシビリティツール|
---------------------------------------
Example:powerhight [-u]
  -a    Antivirus judgment
  -all
        Collect all
  -b    Patch query
  -i    Internet Information
  -r    Registry check
  -s    Get service information
  -u    Get user information and host information
```
